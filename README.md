# student-operations-management-system
A Python-based student operations management system using CSV file handling.

This project is a Python-based system designed to manage basic student operational data
such as attendance and marks. It demonstrates how operational records can be stored,
retrieved, and displayed using CSV file handling.

## Features
- Add and store student records
- Retrieve and display stored data
- CSV-based persistent storage
- CLI version and Google Colab demo version included

## Tools Used
- Python
- Google Colab
- CSV file handling

## How to Run
- Run the CLI version locally using Python
- Run the demo notebook in Google Colab for quick output viewing

## Use Case
This project simulates a basic operations workflow commonly required in academic
and organizational environments.
